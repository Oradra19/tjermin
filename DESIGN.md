
# DESIGN.md

## Design Goal

Create a modern marketplace website inspired by premium ecommerce platforms.

The design should look clean, spacious, elegant, and minimal.

Avoid adding unnecessary decorations.

# Design Priority

1. Match provided UI closely.

2. Keep spacing consistent.

3. Build reusable components.

4. Maintain responsiveness.

5. Smooth micro interactions.

6. Pixel-close implementation.

7. Performance before visual effects.
